# AVIC 科研院所正式汇报 PPT 规范

## 1. 整体风格定位

- **风格**：政务/科研汇报风格，简洁大气、专业严肃，无花哨装饰
- **核心诉求**：信息清晰传递 + AVIC 品牌形象统一
- **核心识别元素**：AVIC Logo、纸飞机+圆环品牌图形、标准品牌蓝贯穿所有页面
- **页面比例**：16:9 宽屏
- **页面结构**：封面 → 目录 → 正文内容（N页）→ 封底（致谢页）

## 2. 配色方案

| 角色 | 颜色 | 用途 |
|------|------|------|
| 主色（品牌蓝） | 深蓝 `#0A2E5C` | 封面/封底背景、目录高亮编号、标题色 |
| 辅助主色 | 青蓝 `#3D9ECF` | 装饰图形、背景纹理、强调色块 |
| 文字色（深底） | 白色 `#FFFFFF` | 封面封底标题正文 |
| 文字色（浅底） | 黑色/深灰 | 内页正文、说明文字 |
| 背景色（内页） | 纯白或极浅灰 `#F5F5F5` | 目录页、正文页底色 |
| 强调色 | 橙色/红色 | 关键词、核心概念标注（克制使用，不超过2种） |
| 次要信息色 | 中灰色 | 次级目录序号、辅助信息 |
| 文本框填充 | 低饱和度浅绿 | 正文文本框背景 |
| 文本框边框/分隔线 | 细浅蓝/浅绿 | 边框、分隔线 |

**背景纹理**：
- 封面/封底：深蓝渐变（上亮下深）+ 半透明网点/点阵纹理
- 内页：纯白或极浅灰底，保持简洁

## 3. 字体规范

| 层级 | 字体 | 字号 | 样式 |
|------|------|------|------|
| 封面主标题 | 微软雅黑 | 48pt | 加粗、白色 |
| 封面副标题 | 微软雅黑 | 32pt | 常规、白色 |
| 内页大标题 | 微软雅黑 | 32-36pt | 加粗、黑色或深蓝色 |
| 内页正文 | 微软雅黑 | 24-28pt | 常规、黑色 |
| 英文/数字 | Arial Unicode MS | 同层级匹配 | 常规 |
| 页脚单位署名 | 微软雅黑 | 12-14pt | 常规、蓝色 |

**文字规则**：
- 一般描述文字用黑色
- 突出核心概念/关键词用橙色/红色暖色调高亮，可搭配加粗
- 所有英文、单位名称、数字统一用 Arial Unicode MS

## 4. 四类页面版式

### 4.1 封面页

- **背景**：全屏深蓝渐变 + 网点纹理
- **水印**：左下角放置半透明（20%-30%）「AVIC 纸飞机+圆环」图形
- **页眉**：右上角 AVIC Logo + 双行单位名称
  - 上行：中航工业西安航空计算技术研究所（白色，微软雅黑常规，18-20号）
  - 下行：AVIC XI'AN AERONAUTICS COMPUTING TECHNIQUE RESEARCH INSTITUTE（白色，Arial Unicode MS，12-14号）
- **标题**：页面居中偏上，主标题白色加粗48号 + 副标题白色常规32号，居中对齐
- **无页码**

### 4.2 目录页

- **背景**：纯白或极浅灰
- **页眉**：右上角小型 AVIC 圆形 Logo
- **左侧**：竖排「目录」二字（微软雅黑加粗，36号，品牌深蓝色）
- **引导线**：1条细蓝色直线（1磅），斜向延伸作为排版参考
- **序号模块**：沿斜线5个矩形编号块，第1块深蓝色+白色数字，其余中灰色+白色数字（微软雅黑加粗24-28号）
- **条目文字**：第1条微软雅黑加粗+深蓝色，其余微软雅黑常规+黑色，统一28号
- **分隔线**：每条目下方细浅绿横线（0.75磅）
- **页脚**：右下角单位全称（蓝色小字）
- **无页码**

### 4.3 正文内容页

- **背景**：纯白或极浅灰
- **页眉**：右上角小型 AVIC 圆形 Logo
- **大标题**：顶部左对齐，微软雅黑加粗32-36号黑色
- **项目符号**：深蓝色实心圆点
- **要点引导词**：微软雅黑加粗深蓝色24-28号，后续描述文字为微软雅黑常规黑色24-28号
- **正文段落**：首行缩进，行间距1.2-1.5倍
- **关键词标注**：橙色/红色暖色调 + 可加粗
- **文本框样式**（二选一）：
  - A：浅绿填充 + 1磅细蓝边框 + 黑色正文
  - B：无填充 + 1磅细蓝边框 + 黑色正文
- **页脚**：右下角单位全称（蓝色小字）

### 4.4 封底（致谢页）

- 与封面页完全镜像一致（背景、页眉、水印全部复制）
- 页面正中央：「感谢倾听！」微软雅黑加粗48-54号白色居中
- 形成首尾视觉闭环

## 5. 装饰元素体系

| 元素 | 形式 | 出现位置 |
|------|------|----------|
| AVIC 圆形徽标 | AVIC 圆框标识 | 所有页面右上角（内页小尺寸，封面封底完整） |
| 纸飞机图形 | 三角形拼接 + 圆环，半透明20-30% | 封面/封底左下角 |
| 斜切矩形块 | 带阴影/边框的编号色块 | 目录页 |
| 细蓝直线 | 1磅，串联序号 | 目录页 |
| 浅绿分隔线 | 0.75磅横线 | 目录页条目间、正文分隔 |
| 网点/点阵纹理 | 半透明叠加 | 封面/封底背景 |

## 6. 幻灯片母版配置

在「视图-幻灯片母版」顶层母版设置全局元素：
- 右上角：小型 AVIC 圆形 Logo
- 右下角：「中航工业西安航空计算技术研究所」蓝色小字（微软雅黑常规12-14号）
- 内页背景：纯白或极浅灰
- 新建2个独立版式：「封面封底版式」「内页正文版式」

## 7. 制作新页面的关键规则

1. 内页继承纯白/极浅灰背景，不在该底色上用深蓝底
2. 所有页面右上角必须保留 AVIC Logo
3. 标题用微软雅黑加粗，正文用微软雅黑常规，英文用 Arial Unicode MS
4. 关键数据/术语用橙色或红色标注强调
5. 整体保持极简克制，内容优先于装饰
6. 默认无页码设计，可按需在页脚添加
7. 强调色不超过2种，颜色仅用于区分信息层级
8. 所有内容严格左对齐或居中对齐，保持页面呼吸感

## 8. 最终检查清单

- 品牌标识：所有页面 Logo、单位名称位置统一，无变形错字
- 字体规范：中文全部微软雅黑，英文/数字 Arial Unicode MS，无混用
- 颜色规范：正文黑色、标题蓝色、强调色仅用橙/红暖色调，无多余杂色
- 对齐规范：左对齐/居中对齐一致，行间距、边距统一
- 视觉闭环：封面与封底风格呼应，整套 PPT 品牌语言一致
